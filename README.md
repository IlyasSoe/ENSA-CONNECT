# 🌐 ENSA Connect

![Bannière du projet](https://github.com/CGI-Cellule-Projets/.github/blob/main/assets/ensa-connect.jpg?raw=true)

## 💡 Idée du projet
ENSA Connect est une plateforme de réseau social professionnel exclusivement dédiée aux étudiants de l'ENSA, actuels et anciens. Elle a pour objectif de créer un écosystème solidaire où les étudiants peuvent trouver des stages, partager leurs expériences de PFE, et bénéficier du mentorat des diplômés maintenant dans la vie active
